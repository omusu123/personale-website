# Ernest Charles - Personal Portfolio

Welcome to my personal portfolio website! This is where I showcase my work, skills, and experience as a Data Analyst.

## Features

- Modern and clean design
- Responsive layout that works on all devices
- Interactive elements and smooth animations
- Multi-language support (English, Spanish, French)
- Accessible interface with keyboard navigation
- Custom 404 error page
- Dark theme by default
- Sections: Home, About, Projects, Contact
- Dark theme

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Technologies Used

- React
- Tailwind CSS
- Framer Motion
- React Router

## Customization

You can customize the website by:
1. Editing the content in `src/App.js`
2. Modifying the color scheme in `tailwind.config.js`
3. Updating the layout and styling in `src/index.css`
